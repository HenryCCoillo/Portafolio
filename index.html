Proyectos

Sistema de Gestion

Desarrollé una aplicación de Gestion, que incluye un sistema de login y registro para garantizar la seguridad de los usuarios. La aplicación cuenta con un potente dashboard de estadísticas que proporciona elrendimiento del negocio. Además, incorpora áreas dedicadas a la gestión de clientes, productos y ventas.

  • Bootstraps
  • Json

Chat en Tiempo Real

Este proyecto tiene como objetivo desarrollar un sistema de chat en tiempo real que permitirá a los usuarios comunicarse de manera instantánea ofreciendo una experiencia de usuario fluida y enriquecedora a través de una plataforma web, utilizando Django Channels.

  • Django
  • WebSocket
  • JavaScript

Factor de Autenticación (2FA)

El proyecto implementé una funcionalidad de Doble Factor de Autenticación (2FA) que refuerza la seguridad de las cuentas. Los usuarios reciben un código único en su correo electrónico después de ingresar su contraseña, asegurando un acceso más seguro.

  • Django
  • 2FA

POKE API Backend

El proyecto Poke API utilizando Django como fuente de datos en el backend. Esta implementado la autenticación de JWT para la seguridad del proyecto y agregué endpoints para listar tanto los Pokémon como sus habilidades correspondientes. Además, documenté el proyecto utilizando Swagger para una mejor comprensión y facilidad de uso.

  • Django
  • Swagger
